Getting Started

To get a local copy up and running follow these simple steps.

Prerequisites

This is the list things you need to highres and how to install them.

  • A Browser (Chrome / Firefox / Safari /Edge)
  • Webdriver

Installation

    pip install highres

Usage

  • If not available already, install a browser such or Chrome or Firefox or Safari or Edge
  • Download webdriver for your Browser
    • Chrome
    • Firefox
    • Edge
    • Safari
  • Add Webdrivers's folder path to system path place your images in a folder, ex: C:\users\dr\Pictures\low-res-images\
